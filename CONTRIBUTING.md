# Contributing

Thanks for your interest in Clouseau! At this stage the project is not accepting code contributions, but bug reports and feature suggestions are very welcome.

## Reporting a bug

Open an issue and include:

- **Clouseau version** (shown in Help → About)
- **Operating system** and Java version if not using the bundled JRE
- **Steps to reproduce** — what you did, what you expected, what happened instead
- **Log file snippet** — a sample of the log format that triggered the issue, if relevant

## Suggesting a feature

Open an issue describing:

- The problem you're trying to solve
- How you'd expect the feature to work
- Any alternatives you've considered

Please check existing issues before opening a new one — your idea may already be tracked.

## Pull requests

Code contributions are not being accepted at this time. This will change in the future.
